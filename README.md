PlayPals: Unplug and Reconnect

PlayPals is a web-based board game rental platform designed to provide an alternative source of entertainment, encouraging users to unplug from screens and reconnect through interactive games. The platform features a customer-friendly catalogue, secure ordering, and a robust admin dashboard for managing games and tracking orders.

Features
- Browse and view board games with images and descriptions
- Add games to cart and select delivery options
- Seamless checkout with payment handling
- Admin dashboard to add, edit, and delete games
- Track orders, manage deliveries, and handle returns
- Responsive design for desktop and mobile

Technologies Used
- Backend: Python (Flask), SQLite  
- Frontend: HTML, CSS, JavaScript  
- Additional Tools: file uploads for game images

Project Highlights
- Designed and implemented full-stack functionality for customer and admin workflows  
- Integrated basic security and data handling practices  
- Built with scalability in mind to support future enhancements
